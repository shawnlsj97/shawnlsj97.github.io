# Portfolio Design References

This folder contains visual references for Shawn's portfolio rebuild. Use these references to guide taste and design judgment, not to copy any specific brand, layout, text, icons, or exact styling.

## Design Principles

- Premium, calm, and restrained.
- Lots of whitespace with deliberate spacing.
- Strong typography with clear visual hierarchy.
- Deep charcoal text on a near-white background.
- Minimal accent color used sparingly for emphasis and interaction.
- Thin borders instead of heavy outlines.
- Subtle depth through soft contrast, layering, and refined surfaces.
- Smooth but minimal interactions.
- Motion should be fast, purposeful, and respectful of `prefers-reduced-motion`.
- The site should feel like a product landing page for Shawn as a senior engineer, not a resume dump.

## Avoid

- Neon gradients.
- Particle backgrounds.
- Heavy shadows.
- Loud animation.
- Template-like Bootstrap feel.
- Student portfolio aesthetic.
- Decorative clutter that competes with Shawn's engineering story.
- Directly copying Apple branding, layout, text, icons, or exact component styling.

## How To Use These References

- Compare new sections against the references for spacing, hierarchy, restraint, and polish.
- Prefer fewer, stronger visual moments over many decorative elements.
- Use reference screenshots to calibrate density, not to reproduce structure.
- When in doubt, simplify the interface and make the content sharper.
